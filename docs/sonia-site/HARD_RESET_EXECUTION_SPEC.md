# 3. Non negotiable visual rules

## Color

Use only this restrained palette direction:

- off white background
- muted lavender gray
- soft gray
- charcoal text
- muted plum accent
- very light lavender surfaces

Do not use:

- beige dominant palette
- cream luxury palette
- gold accents
- spa palette
- influencer palette
- desert neutral palette
- warm tan UI

## Typography

Use exactly two font roles:

1. Editorial serif for display headings only.
2. Clean sans for all body, navigation, buttons, cards, forms.

No random font mixing.

No new decorative fonts.

No heading font inside paragraph content.

No giant body copy.

## Paragraph readability

Body copy must be readable.

Rules:

- body font desktop: 16px to 18px maximum
- body font mobile: 16px
- line height: 1.55 to 1.7
- max paragraph width: 44ch to 52ch
- no paragraph over 2 visual lines in hero
- no paragraph over 3 visual lines in normal sections
- no dense text walls

## Cards and boxes

Use boxes only for:

- service cards
- audience cards
- testimonial cards
- FAQ accordions

Do not put every paragraph in a bordered box.

Do not create alert style green boxes.

Do not use dashboard style cards for reflective copy.
