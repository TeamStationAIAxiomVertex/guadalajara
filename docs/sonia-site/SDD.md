# Sonia McRorey Guadalajara Site SDD

## Objective
Build a modern executive image consulting site for Sonia McRorey that preserves the original psychological sophistication and authorship while improving:

- SEO structure
- readability
- segmentation
- mobile UX
- enterprise credibility
- conversion flow
- semantic clarity

## Critical architecture rule
The original Sonia site voice is the primary system of record.

Do not replace Sonia's voice with SEO language.

## Primary UX problem discovered
The rewritten versions drifted into:

- too much text
- giant paragraph blocks
- semantic over optimization
- invented abstractions
- generic luxury coaching language
- influencer aesthetics
- beige luxury spa palette

This must stop.

## Correct visual direction
The visual system should feel:

- editorial
- executive
- restrained
- intelligent
- modern Guadalajara enterprise
- feminine without becoming fashion influencer
- luxury without appearing flashy

## Approved color system
- muted lavender gray
- soft charcoal
- off white
- cool neutral gray
- subtle muted plum accents

## Forbidden color drift
Do not introduce:

- beige dominance
- gold luxury palette
- warm sand palette
- influencer cream aesthetic
- Dubai luxury styling
- rose gold overload

## Typography direction
Typography should feel:

- editorial
- modern enterprise
- sophisticated
- breathable
- restrained

Avoid:

- trendy luxury influencer fonts
- ultra thin unreadable typography
- excessive contrast between headers and body
- decorative script fonts

## Content segmentation rules
Every section must breathe.

### Hero
Maximum:

- 1 eyebrow line
- 1 headline
- 2 short paragraphs
- 2 CTA buttons
- 1 trust strip

### Content sections
Use:

- cards
- segmented layouts
- icon groups
- whitespace
- visual rhythm
- alternating content/image structure

Avoid:

- essay length sections
- giant blocks of text
- repeated semantic concepts
- stacked keyword clusters

## Page hierarchy
1. Hero
2. Perception problem
3. Core services
4. Who this is for
5. About Sonia
6. Transformation stories
7. Editorial insights
8. FAQ
9. CTA
10. Contact

## SEO requirements
SEO exists to:

- clarify the entity
- clarify the geography
- clarify the service
- support discoverability

SEO must not:

- mutate authorship
- create fake philosophy
- create keyword stuffing
- create repetitive semantic blocks
- invent enterprise jargon

## Geographic anchors

- Guadalajara
- Zapopan
- Andares
- Providencia

- Ciudad de México
- Monterrey

## Primary semantic anchors
- imagen ejecutiva
- presencia profesional
- percepción profesional
- liderazgo

- asesoría de imagen ejecutiva
- consultoría empresarial
- coaching de presencia profesional

## Image direction
Photography should feel:

- real
- understated
- executive
- intelligent
- modern LATAM leadership
- Guadalajara professionals
- enterprise editorial

Avoid:

- exaggerated luxury
- fashion editorial excess
- hyper posed influencer photography
- fake startup stock imagery
- aggressive corporate stock poses

## LLM rewrite rules
Any future AI system modifying the site must:

- preserve Sonia's cadence
- preserve Sonia's restraint
- preserve Sonia's reflective style
- preserve Sonia's psychological tone
- preserve original semantic identity

The AI must not:

- hallucinate new positioning
- invent frameworks
- over explain concepts
- create hype language
- create influencer semantics

## Final quality test
The site should feel like:

An intelligent executive advisor speaking calmly about professional presence.

Not:

A marketing funnel pretending to be Sonia.
